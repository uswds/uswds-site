- **Include a descriptive page title.** Give the 404 page a unique, descriptive `<title>`, like “Page not found.” The page title is often the first thing screen readers announce when a page loads, so a clear title helps users immediately understand that they’ve reached an error page. See [WCAG 2.4.2: Page Titled](https://www.w3.org/WAI/WCAG21/Understanding/page-titled.html) for more information.
